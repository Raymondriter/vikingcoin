# Viking Coin

A website for the Viking Coin cryptocurrency project. Built with HTML, CSS, JavaScript, and PHP, deployed via cPanel.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Deployment:** cPanel (via `.cpanel.yml`)

## Project Structure

- `html/` — HTML pages
- `css/` — Stylesheets
- `js/` — JavaScript files
- `php/` — Server-side PHP scripts
- `images/` — Static image assets
- `index.html` — Main entry point

## Getting Started

### Prerequisites

- A web server with PHP support (e.g., Apache, Nginx)
- Or a local development tool like [XAMPP](https://www.apachefriends.org/) or [MAMP](https://www.mamp.info/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Raymondriter/vikingcoin.git
   ```
2. Place the files in your web server's document root.
3. Open `index.html` in a browser (or access via your local server for PHP functionality).
